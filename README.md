# 🏏 IPL Cricket Analytics Dashboard (2008-2024)

## 📊 Project Overview
Complete end-to-end data analytics project analyzing **17 IPL seasons (2008-2024)** with **260,920 ball-by-ball records**.

## 🛠️ Tech Stack
- **Python (Pandas)** – Data cleaning & feature engineering
- **SQLite** – Database with 33 feature columns
- **Jupyter Notebook** – Interactive analysis
- **Power BI** – Dashboard visualization (coming soon)

## 🚀 How to Use
1. Clone the repository
2. Open `notebooks/01_data_exploration.ipynb` in Jupyter
3. Run all cells to see the analysis
4. Connect Power BI to `data_processed/ipl_clean_data.csv` and `matches_clean.csv`
